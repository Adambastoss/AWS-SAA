**(Amazon RDS Custom for Oracle)**: Este serviço foi projetado **exatamente** para cenários onde você precisa de um banco de dados gerenciado (reduzindo sobrecarga operacional em patches, backups, etc.) mas **requer acesso ao SO subjacente** para personalizações específicas (como certos aplicativos de terceiros ou configurações especiais). Ele permite a configuração de **réplicas de leitura entre regiões** para DR, atendendo aos requisitos de versão mais recente, DR e baixa operação, mantendo o acesso ao SO.
