- **Amazon Aurora MySQL:** ✅ Oferece desempenho superior ao RDS para MySQL, especialmente para cargas de trabalho com muita leitura, devido à sua arquitetura de armazenamento distribuído.

- **Clonagem de Banco de Dados do Aurora (Clone):** ✅Criar um clone do Aurora é uma **operação quase instantânea** que usa um mecanismo de copy-on-write. Ele não coloca carga de leitura adicional no banco de dados de origem (produção) após a criação inicial do ponto de consistência, eliminando a latência para os usuários.
