O **Amazon FSx for Windows File Server** é um serviço de sistema de arquivos totalmente gerenciado que fornece **compartilhamentos de arquivos SMB nativos e altamente disponíveis**, compatíveis com o Windows.

### ==**Amazon FSx File Gateway**==

O **FSx File Gateway** é um appliance local que fornece um cache de baixa latência para um sistema de arquivos do FSx na AWS.
