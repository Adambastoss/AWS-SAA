O **Amazon FSx for Windows File Server** é um serviço de sistema de arquivos totalmente gerenciado que fornece **compartilhamentos de arquivos SMB nativos e altamente disponíveis**, compatíveis com o Windows.  Integra-se perfeitamente com o **Active Directory** da AWS (AWS Managed Microsoft AD)

### ==**Amazon FSx File Gateway**==

O **FSx File Gateway** é um appliance local que fornece um cache de baixa latência para um sistema de arquivos do FSx na AWS.


### ==**Amazon FSx for Lustre:**==

**Lustre** (pronuncia-se _luss-ter_) é um **sistema de arquivos paralelo e distribuído de alto desempenho**, de código aberto, usado principalmente para **computação de alto desempenho (HPC)** e cargas de trabalho intensivas em dados, como simulações científicas, modelagem financeira, processamento de mídia e, como no seu caso, **aplicativos de jogos** que precisam acessar grandes volumes de recursos (texturas, modelos, dados de mundo) com muito baixa latência e alta taxa de transferência.

**Amazon FSx for Lustre:** ✅ Este é **exatamente** o serviço gerenciado da AWS para fornecer um **sistema de arquivos Lustre de alto desempenho e totalmente gerenciado**.

**Compatibilidade Nativa:** ✅ Ele usa o protocolo nativo do Lustre, permitindo que os **clientes Lustre existentes** no seu data center local se conectem a ele diretamente (via VPN ou Direct Connect) sem nenhuma alteração no software do cliente.
