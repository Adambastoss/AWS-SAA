
### **==RUN COMMAND==**

O **AWS Systems Manager Run Command** permite que você execute **comandos personalizados** (como um script de shell `yum update software-x` ou `./install_patch.sh`) em um grande conjunto de instâncias EC2 **imediatamente e de forma simultânea**.

### ==**Patch Manager**==

O **AWS Systems Manager Patch Manager** é excelente para gerenciar patches do **sistema operacional** (Linux, Windows) de forma automatizada e agendada. No entanto, para um **software de terceiros** específico, o Patch Manager pode não ter o repositório de patches pré-configurado.
